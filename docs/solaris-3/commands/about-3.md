# Sonata Effect

## Alias

`son`

## Usage

{% tabs %}
{% tab title="Prefix Command" %}
* `s.sonata <query>`
* `s.son <query>`
{% endtab %}

{% tab title="Slash Command" %}
`/sonata <query>`&#x20;
{% endtab %}
{% endtabs %}

{% hint style="info" %}
You can freely input the query without worrying about entering the exact name since the query will try to find the nearest similar text with your input.
{% endhint %}
